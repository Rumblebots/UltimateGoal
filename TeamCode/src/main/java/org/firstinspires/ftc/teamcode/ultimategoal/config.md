# Controller Mappings
Mappings for the controller.

## Controller 1
- right stick: drive controls
- left stick: drive controls
- right trigger: shifter
- left trigger: shifter

## Controller 2
- right bumper: intake in
- left bumper: intake out
- a button: shoot ring
